# Daily Motivation Dashboard

A clean React web application that fetches random motivational quotes and allows users to save their favorites using browser local storage.

### Live Demo
**https://mid-term-project-motivation.vercel.app/**

---

## Key Features
* **Quote Generation:** Fetches data asynchronously from the ZenQuotes API.
* **Favorite System:** Users can save and remove quotes from a personalized list.
* **Data Persistence:** Favorites are saved in localStorage and persist across page reloads.
* **State Management:** Handles loading, success, and error UI states dynamically.

## Tech Stack
* React (Hooks: useState, useEffect)
* Vite
* JavaScript (ES6+)
* CSS3
